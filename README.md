## Features üåü

- Displays **catgirl images** in your HUD
- coordleaker command for lolz a.k.a cuddle command :)
- player spinner
- crashout command to cool-down after getting killed
- chinese hat (wip)

---

## Installation ‚ö°

1. Make sure you have **Meteor Client** installed  
2. Download the latest `.jar` from the [Releases](https://github.com/turtlegamerw/kawaii-addon-_V2-real/releases) page  
3. Place the `.jar` in your `mods` folder  
4. Launch Minecraft
